## 📌 Project Description

This project presents an interactive Sales Forecasting Dashboard built using Microsoft Power BI.  
It analyzes historical sales data to identify trends, measure performance, and predict future sales using forecasting techniques.

The dashboard enables data-driven decision-making by providing insights into revenue, profit, product performance, and regional distribution.

---

## 🛠️ Tech Stack

**Tools:** Power BI, Microsoft Excel  
**Database:** SQL  
**Programming:** Python  
**Techniques:** Data Modeling, DAX, Data Visualization, Forecasting

---

## 📊 Dashboard Features

- KPI Cards (Total Sales, Total Profit, Forecast Accuracy)
- Monthly Sales Trend Analysis
- Forecasted Sales Visualization
- Top Performing Products
- Region-wise Sales Distribution
- Profit Analysis
