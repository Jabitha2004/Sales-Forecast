# Sales-Forecast
This is data analytics project using Python and Power BI. Python cleans the data and predicts future sales, while Power BI visualizes trends, top products, and regional performance. Perfect for students to learn forecasting, data analysis, and dashboard creation.

## 🛠️ Skills & Technologies Used
- Power BI (Data Modeling, DAX, Dashboard Development)
- SQL (Data Querying)
- Microsoft Excel (Data Cleaning & Analysis)
- Python (Basic Data Processing)
- Data Analysis & Forecasting
